## Data handled by the app

MinimalSudoku is an offline Sudoku app. The app does not require an account,
login, name, email, location, contacts, photos, microphone, camera, or payment
data.

The app may save locally on the device:

- game state, including puzzle, entered numbers, selected cell, difficulty,
  mistakes, move history, and game status;
- audio preferences, volume, and audio mode;
- game sharing links generated by the user.

This data is used only to make the app work, resume a game, remember settings,
and allow manual game sharing.

## Collection and sharing

MinimalSudoku does not automatically send this data to app servers and does not
sell it. The app does not include ads, analytics, user accounts, or in-app
purchases in the version described by this policy.

If the user chooses to share a game, the generated link contains data for the
shared puzzle. Sharing happens only through an explicit user action and may pass
through apps or services chosen by the user.

## Android permissions

The app may declare technical permissions required by the Android runtime or the
Expo/React Native toolchain, such as Internet access, vibration, and storage. In
the version described by this policy, these permissions are not used to collect
personal data on app servers.

## Retention and deletion

Local data remains on the device as long as the app remains installed or as long
as Android keeps the app data. The user can delete it by clearing the app data in
Android settings or by uninstalling the app.

MinimalSudoku does not create user accounts, so there is no account deletion
procedure.

## Security

The data described above is stored locally on the device using app storage.
Because the app does not manage its own backend for this data, no automatic
transmission to app servers is performed.

## Children

MinimalSudoku is not specifically directed to children and does not knowingly
collect personal data from children.

## Changes

This policy may be updated if features, permissions, handled data, or store
requirements change. The updated version should be made available through a
public, active, non-geofenced URL that is not a PDF before submission to Google
Play.
