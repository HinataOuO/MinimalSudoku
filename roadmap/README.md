# MinimalSudoku Roadmap

## Current Goal

Build a lightweight offline Sudoku mobile app with Expo, React Native, TypeScript,
Zustand, Expo Router, NativeWind, and AsyncStorage.

## Macro Tasks

- [x] Create roadmap folder and progress tracking.
- [x] Bootstrap Expo TypeScript app structure.
- [x] Add independent Sudoku engine.
- [x] Add persisted game state.
- [x] Add core Home, Difficulty, and Game screens.
- [x] Add reusable UI components and theme tokens.
- [x] Add initial engine tests.
- [ ] Install dependencies and verify app on device/emulator.
- [ ] Refine UI after first visual run.
- [ ] Expand tests around store and edge cases.

## Milestone 1

Playable offline Sudoku: choose difficulty, generate puzzle, select cells, enter
numbers, validate mistakes, complete puzzle, restart, resume saved game.
